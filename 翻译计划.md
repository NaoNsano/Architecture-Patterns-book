# 翻译计划

## 目标
- 将本书核心正文与附录逐步翻译为中文
- 每完成一个阶段即提交一次（小步提交、可回溯）

## 阶段划分
- [ ] 阶段 1：启动与 Part I 导读翻译
  - 产出：`part1.zh.asciidoc`
  - 提交：完成后立即 commit
- [ ] 阶段 2：前言与导论翻译
  - 产出：`preface.zh.asciidoc`、`introduction.zh.asciidoc`
  - 提交：完成后立即 commit
- [ ] 阶段 3：第一部分章节翻译（Chapter 1~7）
  - 产出：`chapter_01~07_*.zh.asciidoc`
  - 提交：建议按每章或每两章 commit
- [ ] 阶段 4：第二部分与章节翻译（Part II, Chapter 8~13）
  - 产出：`part2.zh.asciidoc`、`chapter_08~13_*.zh.asciidoc`
  - 提交：建议按每章或每两章 commit
- [ ] 阶段 5：尾声与附录翻译
  - 产出：`epilogue_*.zh.asciidoc`、`appendix_*.zh.asciidoc`
  - 提交：按附录分批 commit
- [ ] 阶段 6：统一校对与术语一致性修订
  - 产出：术语统一、链接检查、格式修订
  - 提交：校对完成后 commit

## 翻译约定
- 保留原有 AsciiDoc 结构、锚点、交叉引用与图片标记
- 文件命名采用 `.zh.asciidoc` 后缀，与原文一一对应
- 专有名词首次出现时采用“中文（英文）”形式，后续保持一致
